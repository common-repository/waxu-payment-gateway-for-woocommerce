== Installation ==
For installing the plugin:
1. Be sure to have WooCommerce activated on your wordpress website.
2. On your WP Admin console, Select Plugins->Add new plugin
3. Install "WaXu Payments Gateway"
4. Go to *WooCommerce Settings*
5. Select **payments** and activate WaXu
6. Set your **API Key** and your **API Service** and **Save** your WaXu Settings.
7. Check the box **FETCH COUNTRY LIST**
8. Save *Settings*
